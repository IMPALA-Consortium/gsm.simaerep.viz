SiteList Chart Example

Instructions

This example demonstrates the SiteList chart with gsm.viz site selection functionality:

  • Click on any site in the list to select it
  • Use the dropdown to filter/select sites
  • Selected site is highlighted in blue
  • Selection events are captured and displayed in the info panel

Features Demonstrated

  • Canvas attachment for gsm.kri compatibility
  • Required helper methods (updateConfig, updateSelectedGroupIDs)
  • Group selector with .gsm-widget-control--group class
  • Site selection via click or dropdown
  • Visual highlighting of selected site
  • Custom event dispatching for integration

Integration Points

Container classes: .gsm-widget, .site-list

Canvas chart: canvas.chart attached

Config property: chart.data.config.selectedGroupIDs
